# Lockin

Lockin is a digital student planner designed to help students stay on top of classes, homework, projects, and exams in one organized place.[page:1]

## Features

- Task tracking for homework, projects, and study sessions
- Due dates and priorities so you can see what matters most first
- Overview of upcoming work to reduce last‑minute stress
- Simple, student‑friendly interface focused on clarity and speed

## Why I built this

Managing school work across different apps, paper notes, and emails can get messy fast.[page:1]  
Lockin brings everything into a single planner so it’s easier to remember deadlines and plan ahead.

## Tech stack

## Getting started

### 1. Clone the repository

```bash
git clone https://github.com/Mateus-Barreto-Fernandes/Lockin.git
cd Lockin
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the front‑end/server

In VS Code, open a terminal and run:

```bash
node server/index.js
```

Leave this terminal **open** while you use the app.

### 4. Start the database / dev environment

Open a second terminal in VS Code (same project folder) and run:

```bash
npm run dev
```

Keep this terminal **open** as well.  
After you run `npm run dev`, it should show you a **port number** in the terminal.  
Open that URL in your browser (for example, `http://localhost:3000`) to launch your session.
